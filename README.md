# TPL ISSATSO 2025

A modern web application built with **React** and **Vite** for TPL ISSATSO club (Institut Supérieur des Sciences Appliquées et de Technologie de Sousse). This project serves as a practical lab/course work platform for the 2025 academic year.

🌐 **Live Demo:** [tpl-issatso-2025.vercel.app](https://tpl-issatso-2025.vercel.app)

---

## 🚀 Tech Stack

- **React** — UI component library
- **Vite** — Fast build tool with Hot Module Replacement (HMR)
- **ESLint** — Code linting and quality enforcement
- **Vercel** — Deployment and hosting

---

## 📦 Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

```bash
# Clone the repository
git clone https://github.com/Gueddari-Wassim/TPL-issatso-2025.git

# Navigate to the project directory
cd TPL-issatso-2025

# Install dependencies
npm install
```

### Running Locally

```bash
npm run dev
```

The app will be available at `http://localhost:5173`.

### Building for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
